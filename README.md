# Python Quiz Game

## Overview

This project is a simple command-line Quiz Game built using Python. Players answer a series of general knowledge questions and receive immediate feedback on their answers. The game keeps track of the score and displays the final result at the end.

This project is ideal for beginners learning Python fundamentals such as functions, dictionaries, loops, conditionals, and user input handling.

## How It Works

1. The program stores questions and answers in a dictionary.
2. Each question is displayed to the player.
3. The player enters an answer.
4. The answer is checked against the correct answer.
5. The score is updated accordingly.
6. After all questions are answered, the final score is displayed.
